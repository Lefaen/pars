				Написать программу, которая скачивает все страницы 
				с указанного домена (поддомен считается внешним и не
				должен скачиваться).

 

Входные данные :

                Список доменов для скачивания

                Путь для выходных данных

 

Выходные данные

                Сегментированные по домену файлы с контентом страниц

 

Дополнительные требования:

                Параллельная обработка нескольких доменов

                Обход сайта должен происходить за минимальное количество времени

				Поддержка директивы crawl-delay для сайта

				Запрещается использование third-party библиотек

				Использование TPL и async/await
